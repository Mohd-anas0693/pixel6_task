# pixel6_task
This is Interview task from Pixel6 web Studio Pvt. Ltd.
## Getting Started

To get started with the project, follow these steps:

1. Clone the repository:

    ```bash
    git clone https://github.com/Mohd-anas0693/pixel6_task
    cd pixel6_task
    ```

2. Install dependencies:

    ```bash
    npm install
    ```

3. Run locally:

    ```bash 
        npm run dev
    ```
